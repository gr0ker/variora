# Variora

Веб-приложение для генерации печатных листов с математическими задачами.
Рассчитано в первую очередь на телефон, открывается и с компьютера. Аудитория —
родители и учителя.

**Демо:** https://gr0ker.github.io/variora/

## Что умеет (POC)

Один тип задачи — **примеры на `+ − × ÷` для 4–5 класса**:

- только натуральные числа, деление без остатка, неотрицательные результаты;
- настраиваемые параметры: количество примеров, максимальное число, набор операций,
  перемешивание (чтобы одинаковые операции не шли подряд);
- значения по умолчанию — можно сгенерировать в один клик.

На выходе — **два PDF**: лист с примерами (ученик вписывает ответы прямо в него) и
лист с ответами. Каждая пара получает уникальный код (метка даты-времени) — он попадает
в имена файлов и печатается в шапке листов, поэтому повторные скачивания не затирают
друг друга, а лист ответов легко сопоставить с примерами.

Никакого состояния на сервере: вся генерация и сборка PDF происходят в браузере.

## Технологии

React 18 · Vite · TypeScript · Tailwind CSS · jsPDF (кириллица через встроенный
шрифт DejaVu Sans).

## Разработка

```bash
npm install      # установить зависимости
npm run dev      # дев-сервер Vite
npm run build    # проверка типов + продакшен-сборка в dist/
npm run preview  # локальный предпросмотр сборки
npm test         # юнит-тесты (Vitest)
```

Запуск одного теста: `npx vitest run src/generator/arithmetic.test.ts`.

## Структура

- `src/generator/arithmetic.ts` — генерация примеров (натуральные числа, без остатка,
  всё в пределах максимального числа); покрыта тестами.
- `src/pdf/` — сборка PDF (`buildPdf.ts`), уникальный код пары (`token.ts`),
  кириллический шрифт (`fonts/`).
- `src/i18n/` — слой локализации (`t()` + словарь `ru.ts`), задел под другие языки.
- `src/components/`, `src/App.tsx` — UI формы параметров и кнопки скачивания.

## Деплой

Сайт публикуется на GitHub Pages автоматически при каждом push в `main`
(workflow `.github/workflows/deploy.yml`, источник — GitHub Actions).
В `vite.config.ts` задан `base: '/variora/'` под подпуть Pages.
